Ручная загрузка стикерпаков (без скрипта импорта)
================================================

Где лежат файлы, которые отдаёт сервер
--------------------------------------
Корень:  server/uploads/stickers/
Правило: один стикерпак = одна подпапка (латиница, цифры, подчёркивание — без пробелов).

Пример:
  server/uploads/stickers/soviet_posters/
  server/uploads/stickers/memes_2026/

Внутри каждой папки обязательно:
  - manifest.json  — список стикеров для клиента (см. пример в _example/)
  - файлы картинок/анимаций, имена как в manifest (поле "file")

Публичный URL (как у аватаров и чата):
  /uploads/stickers/<имя_папки>/<файл>

Каталог server/sticker-packs/ в репозитории
--------------------------------------------
Сюда можно класть копии паков для бэкапа или правок в git; на сервере их нужно
скопировать в server/uploads/stickers/ (или смонтировать тот же volume).

Папка server/uploads/ в .gitignore — бинарники в коммит не попадают.

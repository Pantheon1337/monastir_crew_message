# Monastir Crew Message (MCM)

> Minimalist social network for crews. No likes counter, no algorithms.  
> Chat 1-1 | Group rooms | Stories | Timeline  
> **Brutalist UI · Silence Mode · Anti-pafos**

---

## 🧭 Roadmap

### Milestone 0: Project Foundation & Architecture
- [ ] M0.1 Инициализация монорепозитория (client + server + shared types)
- [ ] M0.2 Настройка TypeScript + ESLint + Prettier
- [ ] M0.3 Docker Compose (PostgreSQL + Redis + MinIO для медиа)
- [ ] M0.4 WebSocket сервер (Node.js + Socket.IO или uWebSockets)
- [ ] M0.5 Простая модель пользователя (регистрация по invite-коду)
- [ ] M0.6 Базовая аутентификация (JWT + refresh token)

### Milestone 1: Chat Core (1-1, Groups, Rooms)
- [ ] M1.1 Личные чаты (1-1) — создание, отправка текста
- [ ] M1.2 Группы (2+ пользователей) — создание, добавление участников
- [ ] M1.3 Комнаты (паблики, как в ТГ) — публичная история, участники без ограничений
- [ ] M1.4 Редактирование/удаление сообщений
- [ ] M1.5 Реакции (только 🔥 — один смайл, без выбора)
- [ ] M1.6 Индикатор «печатает...» (одна серая точка)
- [ ] M1.7 Упоминание всей комнаты через `@crew`
- [ ] M1.8 История сообщений (пагинация, загрузка при скролле)

### Milestone 2: Stories (аналог ТГ + Инстаграм)
- [ ] M2.1 Создание истории (только текст + сырое фото без обрезки)
- [ ] M2.2 Отображение сторис (круглые аватарки с прогресс-баром сверху ленты)
- [ ] M2.3 Просмотр истории (тап — следующий кадр, холд — пауза)
- [ ] M2.4 Автоудаление через 24 часа
- [ ] M2.5 Опция «Оставить в ленте» (автоматический пост после 24ч)
- [ ] M2.6 Реакция на историю (только 🔥 в личку автору)

### Milestone 3: Feed (аналог VK — без лайков)
- [ ] M3.1 Создание поста (текст + фото)
- [ ] M3.2 Лента (только хронологическая, никакой ML)
- [ ] M3.3 Комментарии к посту (вложенные, без подсчета)
- [ ] M3.4 Репост («Эхо») — перенос в свою стену без цитирования
- [ ] M3.5 Лайк без счетчика (иконка огня, цифра скрыта)
- [ ] M3.6 Прикрепление поста из комнаты в ленту

### Milestone 4: Unique Feature — Silence Mode (Тишина)
- [ ] M4.1 Переключатель «В дозоре» (глобальный или на время)
- [ ] M4.2 При включении: лента → серая градация, чаты → без звука, истории → скрыты
- [ ] M4.3 Поле ввода «Что случилось, командир?» (только экстренные сообщения в активную комнату)
- [ ] M4.4 Таймер выхода из режима (1ч, 4ч, 8ч)
- [ ] M4.5 Push-уведомления о важном (настраивается пользователем)

### Milestone 5: UI/Polish — Brutalist & Anti-Pafos
- [ ] M5.1 Цветовая схема: фон `#1a1c20`, текст `#e0e0e0`, акцент `#c17b4b` (ржавый)
- [ ] M5.2 Шрифт: JetBrains Mono / Inter (моноширинный в чатах)
- [ ] M5.3 Темная тема по умолчанию (без переключения на светлую)
- [ ] M5.4 Адаптив: мобильная версия (приоритет) + десктоп (узкие колонки)
- [ ] M5.5 Анимации: только плавное появление (никаких упругих эффектов)
- [ ] M5.6 Отказ от градиентов, теней и 3D

### Milestone 6: Production Ready
- [ ] M6.1 Регистрация только по инвайтам (закрытый клуб)
- [ ] M6.2 Хранение медиа в MinIO / S3
- [ ] M6.3 WebSocket reconnection с экспоненциальной задержкой
- [ ] M6.4 Логирование ошибок (без сбора личных данных)
- [ ] M6.5 Документация API (OpenAPI / Swagger)
- [ ] M6.6 Инструкция по деплою (docker-compose production)
- [ ] M6.7 Домен + HTTPS (Let's Encrypt)

|------|------------|

| Mobile | PWA (в перспективе React Native) |
| Backend | Node.js + Express + Socket.IO |
| DB | PostgreSQL + Redis |
| Media | MinIO / S3 |

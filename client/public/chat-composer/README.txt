Иконки панели ввода чата (PNG с прозрачностью, ~24–32 px по большей стороне)

Файлы в этой папке с такими именами:

  file.png    — скрепка / вложение
  sticker.png — стикеры и эмодзи
  video.png   — режим видеокружка
  mic.png     — режим голоса / микрофон

После добавления или замены файлов:
  1) Пересоберите клиент: npm run build (vite копирует public/ в dist/).
  2) Чтобы браузер подхватил новые PNG, в client/.env задайте, например:
       VITE_CHAT_COMPOSER_ICONS=2
     и увеличивайте число при каждой замене иконок, затем снова npm run build.

Если файла нет — в интерфейсе останутся запасные символы (эмодзи).
